## определение
это протоколы и системы, которые обеспечивают обмен данными между собой.
современные службы работают по модели TCP/IP (пакетная передача данных)
## ключевые принципы
1. клиент-серверная модель, где клиент делает запрос, а сервер на него отвечает
2. порты - каждая служба использует собственный порт для обмена запросами
3. безопасность - почти у каждой службы есть защищенная версия
## основные службы
1. DNS (система доменных имен) - переводит удобные имена сайтов в IP-адреса
2. HTTP/HTTPS (гипертекстовый протокол передачи данных) - основная служба для работы веб-сайтов, через неё осуществляется передача данных: запросов от клиента и ответов от сервера. HTTPS - защищенная версия с шифрованием.
3. SMTP (простой протокол передачи почты) - протокол для отправки Е-мэйл, является почтальоном, который доставляет письма от сервера к получателю
