## введение
цикл это конструкция, позволяющая выполнять определённую задачу некоторое кол-во раз. в питон есть несколько реализаций: для диапазона значений и для условия. первый завершится сразу после перебора всех значений в диапазоне, а второй - когда условие станет ложным

## цикл for
реализация первого вида осуществляется через следующую конструкцию:
```python
for i in <диапозон_значений>:
	инструкция
	инструкция
```
после ключевого слова **for** следует произвольное имя для локальной переменной, которая будет использоваться внутри цикла. ей последовательно будут присваиваться значения из уникального диапазона. диапазон может быть любой структурой данных, которая может быть итерирована - (списки, строки) также можно использовать функцию для генерации числовых последовательностей, **range**(конец) или **range**(start, end, step).в но результате её выполнения будет сгенерирован список, состоящий из чисел, который сразу же можно передать в переменную цикла. Передача значений из диапазона осуществляется при помощи оператора in
также есть операторы взаимодействия с циклами, такие как **break, continue, else**.
**break** - оператор прерывания выполнения цикла
**continue** - пропуск текущей итерации цикла, весь код после этого оператора будет проигнорирован
**else** - блок, выполняемый после завершения цикла
