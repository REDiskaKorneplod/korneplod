
в питоне существует условный оператор или оператор ветвления. он позволяет управлять потоком выполнения программ при помощи реализации механизма динамического ветвления на основе оценки булевых выражений. if  и elif опираются на истинность или ложность логического выражения и выполняет соответствующий код блока.
## синтаксис и структура

блок if "если":
```python
if <условие>:
	<блок_констр1>
	<блок_констр2>
	<блок_констр3>
```
если условие выполняется, то тогда выполняется и код внутри блока
блок кода внутри оператора if всегда начинается с отступа, который равен 4 пробелам или 1 табу от родительского if 
связка if-elif-else "если, если иначе, иначе":
```python
if <ycl1>:
	<block_konstr1>
elif <ycl2>:
	<block_konstr2>
else:
	<block_konstr3>
```
в данном коде добавлена проверка альтернативных условий. она срабатывает в случае, если не выполнились предыдущие условия.
также появился не условный блок, который срабатывает, если все условия перед ним были оценены как ложные.
