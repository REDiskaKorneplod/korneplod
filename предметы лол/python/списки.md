## определение
списки - упорядоченные изменяемые коллекции произвольных типов, они создаются при помощи [ ], или функции - конструктора list(). к каждому элементу можно обратиться по его индексу, индексация начинается с нуля
списки необязательно могут содержать однотипные объекты.
```Python
o = []
a = list()
oa =["spider",True,3]
ao = list(oa)
```
## функция list
принимает значения итерируемого типа, т е другие списки, строки и тд
## срезы
способ получения части списка при помощи указания индексов, готовый срез является новым списком. для создания среза указывается имя списка и определённые индексы. всего есть 3 способа указать правило генерации среза:
1. указать конечное значение - будет сгенерирован срез от 0 до указанного
2. указать начальное и конечное значение
3. указать начальное и конечное значение и шаг.
если требуется использовать 1 из указанных способов, но отсутствует некоторая информация, в [ ] можно поставить : вместо пропущенного значения
## методы работы со списками
- append(item) - добавить - сначала указывается имя списка, затем через точку вызывается метод append, в который передается элемент item для добавления, появляется в конце списка -  `abc.append("1")` 
- index(item) - индекс конкретного элемента - сначала указывается имя списка, через точку вызывается метод index, в который передается элемент item. в результате работы возвращает индекс указанного элемента. если элементов с одинаковым значением несколько, возвращает индекс первого найденного в тексте, если нет, выбрасывает исключение в ValueError
- len(list) - длина списка - сначала вызывается функция len, а ей передается конкретный список, возвращает длину списка в виде целого числа
- min() и max() - возвращают мин или макс значение значение из указанного в качестве аргумента, списка
- sorted() - возвращает новый отсортированный список из элементов любого итерируемого объекта, не изменяя исходный. Для сортировки списка на месте (изменяя его) используется метод .sort()
- insert() - добавляет элемент в список на указанную позицию, сдвигая остальные элементы вправо. Синтаксис `list.insert(index, element)` позволяет указать индекс (`index`) и сам элемент (`element`), который нужно вставить. Например, `list.insert(0, 'item')` добавит `item` в начало списка, а `list.insert(2, 'item')` вставит его на позицию с индексом 2.
- extend() -  это метод списков в Python, который добавляет элементы из итерируемого объекта (списка, кортежа, строки и т. д.) в конец существующего списка. В отличие от `append()`, который добавляет один элемент как единое целое, `extend()` добавляет каждый элемент из переданного объекта по отдельности, изменяя исходный список на месте и возвращая `None`.
- remove() - удаляет существующий в списке item
- pop() - используется для удаления и возврата элемента из списка. Если указать индекс, удаляется и возвращается элемент по этому индексу; если индекс не указан, удаляется и возвращается последний элемент списка. Важно помнить, что `pop()` изменяет исходный список на месте.
- count() - это метод и функция для подсчёта чего-либо. Чаще всего используется метод `count()`, который подсчитывает количество вхождений элемента в последовательности, такой как список, строка или кортеж. Также существует функция `count()` из модуля `itertools` для создания бесконечного итератора.
- reverse() - Для реверса (обратного порядка) списка в Python есть несколько способов: `list.reverse()` **изменяет список на месте**, возвращая `None`; срезы `[::-1]` создают **новый** перевернутый список, не изменяя оригинал; а встроенная функция `reversed()` возвращает **итератор**, который нужно преобразовать (например, в `list()`) для получения нового списка.
- copy() - Для копирования списков в Python можно использовать разные методы, которые создают поверхностную или глубокую копии. Поверхностное копирование создает новый список, но вложенные объекты остаются ссылками на оригинал. Глубокое копирование полностью дублирует все объекты, включая вложенные.
