## определение
списки - упорядоченные изменяемые коллекции произвольных типов, они создаются при помощи [ ], или функции - конструктора list(). к каждому элементу можно обратиться по его индексу, индексация начинается с нуля
списки необязательно могут содержать однотипные объекты.
```Python
o = []
a = list()
oa =["spider",True,3]
ao = list(oa)
```
## функция list
принимает значения итерируемого типа, т е другие списки, строки и тд
## срезы
способ получения части списка при помощи указания индексов, готовый срез является новым списком. для создания среза указывается имя списка и определённые индексы. всего есть 3 способа указать правило генерации среза:
1. указать конечное значение - будет сгенерирован срез от 0 до указанного
2. указать начальное и конечное значение
3. указать начальное и конечное значение и шаг.
если требуется использовать 1 из указанных способов, но отсутствует некоторая информация, в [ ] можно поставить : вместо пропущенного значения
## методы работы со списками
- append(item) - добавить - сначала указывается имя списка, затем через точку вызывается метод append, в который передается элемент item для добавления, появляется в конце списка -  `abc.append("1")` 
- index(item) - индекс конкретного элемента - сначала указывается имя списка, через точку вызывается метод index, в который передается элемент item. в результате работы возвращает индекс указанного элемента. если элементов с одинаковым значением несколько, возвращает индекс первого найденного в тексте, если нет, выбрасывает исключение в ValueError
- len(list) - длина списка - сначала вызывается функция len, а ей передается конкретный список, возвращает длину списка в виде целого числа
- min() и max() - возвращают мин или макс значение значение из указанного в качестве аргумента, списка
