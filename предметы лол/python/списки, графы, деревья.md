# классы
пользовательский тип данных, конструкция ООП (конструкция объекто-ориентированного программирования), которая описывает тот или иной объект, его свойства и способы взаимодействия с ним
класс имеет поля - переменные, которые будут принадлежать к его экземпляру, а также методы - функции, которые также будет можно использовать для его экземпляра
класс - шаблон для реализации какого-либо объекта. при помощи класса мы описываем абстрактный  глав объект, а экземпляры будут являться настоящими, конкретными объектами  с характеристиками и действиями глав объекта
класс решает несколько важных задач - дублирование кода, унификации объектов
## классы в питоне
объявляются при помощи `class` и указании имени класса, затем ставится : и со следующей строки описывается тело класса

чтобы объявить поля класса используется ключевое слово `self`, после которого, через точку указывается имя поля. поле тоже что и переменная, но внутри класса. однако через `self` к переменной можно обращаться только через метод тела. в параметрах метода также обязательно должен быть указан `self`.

для инициализации полей при создании экземпляра класса используется __ `inil` __ . она обязательно принимает параметр `self`. эта функция будет автоматически вызвана при создании экземпляра класса. если она не объявлена, будет использован конструктор по умолчанию (пустой)

создание экземпляра класса (объект класса, объект) происходит вне тела класса, при помощи указания имени класса и круглых скобок после имени. если в конструкторе self  указаны другие параметры, то при создании объекта в скобках мы также укажем атрибуты - то, что является параметрами при описании функции.
