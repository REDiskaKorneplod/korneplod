## теория множеств
множество - коллекция уникальных элементов (не повторяются), множества могут быть бесконечными, а также имеют мощностью. в rython есть специальная структура данных - set (является дословным переводом множества)

## свойства множеств
- коммутативность: А in B == B in A
- ассоциативность: (A in B) in C = A in (B in C)
- дистрибутивность:  a in not(B and C) = (A and B) in (a and C)

Аксиома Цермело-Френкель (ZFC) - набор правил для построения множеств, чтобы избежать парадоксов, например "множества всех множеств"

## комбинаторика

раздел математики, которая решает задачи о подсчете числа способов выбора, размещения или упорядочивания объектов из конечного множества
### основные понятия
- работа с множествами или к-объектами из них
- принцип умножения: если событие А можно выполнить m способами, а событие В n способами, и при этом они не зависимы, то вместе кол-во способов = m * n
- принцип события, если события исключают друг друга, то общее кол-во способов - сумма
### примеры и 3 фундаментальные формулы
факториал:
	его формула: n! = n * (n - 1)... * 1
	он используется, когда нужно переупорядочить все объекты, т.е. найти возможное кол-во способов это сделать
метод перестановки:
	 P(n,k) = n! / (n - k)!
	 мы находим кол-во перестановок - уникальных размещений k-объектов в n-множестве
сочетания:
	 C(n,k) = n! / k! * (n-k)! = P(n,k) / k!
	 позволяет определить кол-во сочетаний или комбинаций для указанного кол-ва элементов k в множестве n
